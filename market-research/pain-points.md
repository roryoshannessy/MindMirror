# User Pain Points

## Top 10 Raw Pain Points

### 1. Blank Page Paralysis
**Real quote:** "Every time I open one, I see the blank page and just close it."
- Friction at first moment of use
- Users don't know what to write
- Causes immediate abandonment (day 1)
- **Impact:** Kills acquisition

### 2. Journaling Feels Like Homework
**Real quote:** "Expect you to 'think deeply' on demand, which feels unnatural."
- Association with school/obligation
- Guilt if missed
- Pressure to be "good at" journaling
- **Impact:** Creates avoidance behavior

### 3. Quick Burst, Then Forgotten
**Real quote:** "Excited for 3 days, forget it exists by day 7."
- No reminder system needed (users hate them)
- No habit formed because initial motivation fades
- Requires active decision to return
- **Impact:** 70% churn by week 2

### 4. Long Entries = Too Much Effort
**Real quote:** "I want to dump quickly. 30 seconds, not 5 minutes."
- Users don't want to compose
- Writing itself is friction
- Prefer audio/quick capture
- **Impact:** Drives users to simpler apps (Daylio)

### 5. Apps Feel Too Formal or "Dear Diary"
**Real quote:** "Very 'dear diary' and formal, which makes me overthink what I'm writing."
- Privacy anxiety (someone might read this)
- Self-consciousness about tone
- Pressure to be articulate
- **Impact:** Users don't write honestly

### 6. Mood Tracking Becomes Obsessive
**Real quote:** "Constantly asking what mood I was in, then 'preempting' my mood to fit the chart."
- Self-surveillance anxiety
- Creates meta-worry ("why am I depressed again?")
- Quantifying the unquantifiable backfires
- **Impact:** Increases anxiety instead of reducing it

### 7. No Insight From Data
**Real quote:** "Wrote 'today was hard' and closed app feeling exactly the same."
- Journaling feels pointless
- No transformation happens
- Just data collection, no analysis
- **Impact:** Users see it as "talking to a wall"

### 8. Can't Return After Gap
**Real quote:** "Should I address the gap, start fresh, or ignore it?"
- Shame around quitting
- Awkwardness returning
- No "comeback" experience
- **Impact:** Users who churn don't come back

### 9. Privacy Fear Is Intense
**Real quote:** "Never felt more violated. Now questions what would happen if anyone read it."
- AI analysis triggers privacy concern
- Data ownership anxiety
- Breach fear
- **Impact:** Subscription distrust premium

### 10. Subscription Distrust (Critical)
**Real quote:** (From Trustpilot reviews) "Charged $30/month, couldn't cancel"
- Auto-renewal skepticism
- Cancellation friction complaints
- Refund denial stories
- **Impact:** 15-20% trust tax on mental wellness subscriptions

## Emotional Triggers (By Pattern)

| Pain Point | Triggers | User State | Action |
|---|---|---|---|
| Blank page | Paralysis | Overwhelm | Close app |
| Homework feeling | Guilt + obligation | Avoidance | Don't open |
| Forgotten by day 7 | Habit failure | Shame | Uninstall |
| Too much effort | Exhaustion + friction | Busy | Switch apps |
| Too formal | Self-consciousness | Guarded | Write less |
| Obsessive tracking | Anxiety + meta-worry | Worse | Stop using |
| No insight | Meaninglessness | Hopelessness | Quit |
| Can't return | Shame + awkwardness | Stuck | Don't reinstall |
| Privacy fear | Violation | Distrust | Don't share |
| Subscription trap | Anger + distrust | Skeptical | Avoid trials |

## Language Patterns (Real User Voice)

"I spend more time setting up the perfect template than actually journaling."
- Perfectionism trap

"It's like talking to a wall."
- Lack of reciprocity

"I was depressed. Today I logged depressed. Tomorrow probably depressed."
- Groundhog Day effect

"No writing required" (why Daylio wins)
- Friction is everything

"Gives me permission to be messy."
- Safety/authenticity matters

"I need someone to read this and push back."
- Accountability desire (but competitor gap)

## Impact Ranking (By Conversion Loss)

1. **No clear outcome** (blank page, no insight) → 40% drop
2. **High friction entry** (typing, effort) → 30% drop
3. **Shame/guilt mechanics** (missed days, judgment) → 25% drop
4. **Privacy anxiety** (data sharing concerns) → 20% drop
5. **Subscription distrust** (billing skepticism) → 15% drop
