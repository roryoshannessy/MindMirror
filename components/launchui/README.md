# LaunchUI (deferred)

MindMirror’s marketing shell currently uses custom sections and shadcn primitives instead of the [LaunchUI](https://launchuicomponents.com/) registry blocks described in `FOUNDATION.md` M1.

If we adopt LaunchUI later, add components under this directory via the LaunchUI + shadcn registry flow and refactor the homepage sections. Until then, treat M1 “LaunchUI” as **formally deferred** so the playbook matches the repo.
